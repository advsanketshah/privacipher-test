Clauselab: Client-Side Contract Lifecycle Management

A zero-backend CLM workspace. Track contracts, browse a curated clause library, assemble drafts from templates, and run AI-assisted review with your own API key. No accounts, no servers, no telemetry. All data sits in the user's browser.

  • Contracts Repository: Metadata, status, renewal dates, tags, notes. Full CRUD with filtering, search, and one-click JSON backup and restore.
  • Clause Library: 45+ practitioner-drafted clauses across 19 categories including DPDPA 2023, GDPR Article 28, indemnification, limitation of liability, and boilerplate. Users can add their own.
  • Template Drafter: Form-based assembly with live preview. Mutual NDA, SaaS Subscription, Consulting Services, and Data Processing Addendum templates. Exports to Word, markdown, and plain text.
  • AI-Assisted Review (BYOK): Five review modes covering issue spotting, redline suggestions, plain-English summary, DPDPA+GDPR compliance check, and missing-clauses check. Works with Gemini, OpenAI, or Anthropic keys.
  • Zero-Trust Architecture: IndexedDB storage, localStorage for settings, no analytics, no trackers, no third-party scripts beyond fonts. API keys never leave the device except when the user runs a review.
